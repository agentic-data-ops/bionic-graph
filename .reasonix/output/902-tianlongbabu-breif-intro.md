## 第1章：核心人物

### 1.1 乔峰（萧峰）

契丹人，本姓萧，幼年被乔氏夫妇收养。丐帮前任帮主，武功盖世，擅降龙十八掌。因身世秘密被揭露而离开丐帮，后成为辽国南院大王。最终因阻止宋辽战争而自尽于雁门关。

**关联人物**：
- 父亲：萧远山（未死，隐居少林三十年）
- 义兄：段誉（结拜兄弟）
- 义弟：虚竹（结拜兄弟）
- 爱人：阿朱（被乔峰误杀）
- 仇人：马夫人康敏（揭露身世，挑拨离间）
- 对手：慕容复（多次交手）
- 恩人：玄慈方丈（养父之师？实为带头大哥）

### 1.2 段誉

大理国镇南王段正淳之子，实为段延庆与刀白凤所生。天性纯善，厌恶武功，却因奇遇学会北冥神功、凌波微步、六脉神剑。

**关联人物**：
- 生父：段延庆（四大恶人之首）
- 养父：段正淳（镇南王，实为伯父）
- 母亲：刀白凤（摆夷族女子）
- 爱人：王语嫣（最终与段誉在一起）
- 义兄：乔峰（结拜大哥）
- 义兄：虚竹（结拜二哥）
- 姐妹：木婉清（同父异母）、钟灵（同父异母）、阿朱（同父异母）、阿紫（同父异母）
- 师父：鸠摩智（被迫学武，实则仇人）
- 恩人：枯荣大师（传授六脉神剑）

### 1.3 虚竹

少林寺小僧，相貌丑陋，朴实善良。因机缘巧合破解珍珑棋局，得无崖子传功，成为逍遥派掌门。后任灵鹫宫宫主，娶西夏公主李清露。

**关联人物**：
- 生父：玄慈方丈（少林寺方丈）
- 生母：叶二娘（四大恶人之二）
- 义兄：乔峰（结拜大哥）
- 义兄：段誉（结拜三弟）
- 师父：无崖子（逍遥派掌门，授毕生功力）
- 师父：天山童姥（授灵鹫宫武学）
- 爱人：李清露（西夏公主）
- 师兄：丁春秋（星宿老怪，无崖子逆徒）
- 管家：梅兰竹菊（灵鹫宫四婢）

### 1.4 慕容复

姑苏慕容氏家主，鲜卑族后裔，立志复兴大燕国。武功"以彼之道还施彼身"，与乔峰并称"北乔峰南慕容"。最终因复国梦碎而疯癫。

**关联人物**：
- 父亲：慕容博（假死，幕后策划雁门关事件）
- 丫鬟：阿朱（慕容家婢女，后被乔峰所救）
- 丫鬟：阿碧（始终跟随慕容复）
- 爱慕者：王语嫣（表妹，慕容复始终利用她）
- 对手：乔峰（一生之敌）
- 盟友：鸠摩智（互相利用）
- 下属：风波恶、邓百川、公冶乾、包不同（四大家将）

## 第2章：次要人物

### 2.1 辽国阵营

- **萧远山**：乔峰生父，三十年前雁门关之战的受害者，假死后潜伏少林三十年，偷学武功，复仇元凶。
- **阿朱**：段正淳与阮星竹之女，慕容家婢女，精通易容术。乔峰恋人，被乔峰误杀于小镜湖畔。
- **阿紫**：段正淳与阮星竹之女，阿朱之妹，刁蛮狠毒。痴恋乔峰，最终为救乔峰而自杀。

### 2.2 大理（段氏）阵营

- **段正淳**：大理镇南王，风流成性，情人遍布天下，段誉养父。
- **段延庆**：四大恶人之首，大理前太子，因政变流亡。段誉生父。
- **刀白凤**：段正淳之妻，摆夷族，因报复段正淳而与段延庆一夜情，生段誉。
- **枯荣大师**：天龙寺高僧，修炼枯荣禅功，传授段誉六脉神剑。

### 2.3 逍遥派系

- **无崖子**：逍遥派掌门，被逆徒丁春秋暗算，全身瘫痪，隐居擂鼓山。
- **天山童姥**：灵鹫宫主人，无崖子师姐，练八荒六合唯我独尊功，每三十年返老还童。
- **李秋水**：无崖子师妹，西夏皇太妃。
- **丁春秋**：星宿派掌门，无崖子逆徒，擅用毒与化功大法，人称星宿老怪。

### 2.4 西夏

- **李清露**：西夏公主，虚竹之妻，李秋水孙女。
- **西夏皇帝**：李清露之父。

### 2.5 四大恶人

- **段延庆**（恶贯满盈）：四大恶人之首
- **叶二娘**（无恶不作）：虚竹之母，因儿子被夺而精神失常
- **南海鳄神**（凶神恶煞）：段誉之师（名义上）
- **云中鹤**（穷凶极恶）：好色之徒

### 2.6 其他

- **鸠摩智**：吐蕃国师，痴迷武学，为六脉神剑不择手段。
- **康敏（马夫人）**：段正淳旧情人，丐帮马大元之妻，因恨报复，揭露乔峰身世引线。
- **玄慈方丈**：少林寺方丈，三十年前雁门关之战带头大哥，虚竹生父。
- **扫地僧**：少林藏经阁无名老僧，佛法武功深不可测，化解萧远山父子与慕容博父子的恩怨。

## 第3章：主要事件与因果关系

### 3.1 导火索 — 三十年前的雁门关之战

**起因**：慕容博假传消息，称契丹武士将入侵少林抢武学秘籍，诱使中原武林在雁门关伏击契丹平民。

**参与者**：玄慈（带头大哥）、汪剑通（丐帮帮主）、萧远山夫妇及契丹族人。

**结果**：
- 萧远山之妻被杀，萧远山悲愤跳崖（实则未死）
- 萧远山之子被乔氏夫妇收养，即后来的乔峰
- 玄慈、汪剑通等人得知真相后愧疚，立誓保密

**因果链**：
```
慕容博的野心 → 假传消息 → 雁门关伏击 → 萧远山妻死 → 乔峰成为孤儿
                                                         ↓
                                                   三十年后身世曝光
                                                         ↓
                                                   马夫人揭露 → 乔峰离丐帮 → 寻亲复仇
```

### 3.2 乔峰身世之谜与丐帮事变

**事件**：乔峰继任丐帮帮主后，马夫人康敏因段正淳抛弃而迁怒于所有男人，设计揭露乔峰契丹人身份。

**因果**：
```
马夫人 → 诱导马大元揭露真相 → 马大元不从 → 马夫人勾结白世镜杀马大元
    ↓
杏子林大会 → 全冠清带领丐帮众人逼宫 → 乔峰离开丐帮
    ↓
乔峰追查身世 → 误以为段正淳是带头大哥 → 小镜湖误杀阿朱
```

### 3.3 小镜湖悲剧

**起因**：
1. 乔峰追查"带头大哥"，误以为段正淳是仇人
2. 阿朱得知自己是段正淳之女，陷入两难
3. 阿朱易容为段正淳赴约

**关键转折**：
```
阿朱求乔峰："我求你一件事——你无论如何要答应我"
乔峰："你吩咐吧，我一定办到"
阿朱："你答应我，从此不再找我爹爹（段正淳）报仇"
```

**悲剧结果**：雷雨之夜，乔峰一掌打死易容为段正淳的阿朱。

**连锁反应**：
```
阿朱之死 → 乔峰终生愧疚 → 发誓照顾阿紫
                                  ↓
                        阿紫爱上乔峰 → 纠缠一生
```

### 3.4 珍珑棋局 — 命运的交汇点

**事件**：逍遥派无崖子在擂鼓山设珍珑棋局，寻衣钵传人。

**参与者**：段誉、虚竹、慕容复、鸠摩智、丁春秋等。

**结果**：虚竹被鸠摩智逼迫之下胡乱落子，阴差阳错破解珍珑棋局。

**因果链**：
```
虚竹破解珍珑 → 无崖子传功 → 虚竹成为逍遥派掌门
                                    ↓
天山童姥找虚竹 → 传授灵鹫宫武学 → 虚竹掌灵鹫宫
                                    ↓
西夏招亲 → 虚竹与李清露成婚 → 虚竹身份地位飞跃
```

### 3.5 少室山大战 — 全书高潮

**地点**：少林寺

**参与方**：
- 乔峰、段誉、虚竹三兄弟联手
- 萧远山现身，揭开三十年真相
- 慕容复、鸠摩智、丁春秋为敌

**关键对决**：
1. **段誉 vs 慕容复**：六脉神剑击败慕容复，确立段誉武功大成
2. **虚竹 vs 丁春秋**：逍遥派内斗，虚竹清理门户
3. **乔峰 vs 慕容复、游坦之、丁春秋**：乔峰以一敌三，尽显英雄本色

**真相揭露**：
```
萧远山现身 → 指认玄慈为带头大哥 → 揭露叶二娘与玄慈私情
    ↓                              ↓
玄慈认罪受杖 → 叶二娘殉情   虚竹得知父母身份
    ↓
扫地僧现身 → 化解萧远山与慕容博的仇恨 → 二人出家
```

### 3.6 雁门关结局 — 悲剧的终结

**起因**：辽帝耶律洪基要乔峰出兵攻宋，乔峰不从。

**事件**：
1. 乔峰被辽帝囚禁
2. 段誉、虚竹率群雄救出乔峰
3. 乔峰挟持辽帝，逼其立誓永不攻宋

**最终结局**：
```
乔峰胁迫辽帝立誓 → "我有何面目见天下英雄" → 断箭自尽
                                                    ↓
阿紫抱乔峰尸体跳崖 → 游坦之随阿紫跳崖 → 三人同葬
                                                    ↓
段誉回大理继位 → 虚竹掌灵鹫宫 → 慕容复发疯
```
