# Plan 004: 前端图谱可视化界面

## 技术选型

| 层 | 选择 | 说明 |
|----|------|------|
| 框架 | **React 18** | Vite 脚手架 |
| 图谱可视化 | **vis-network** | 力导向布局、双击事件、物理仿真 |
| 样式 | **Tailwind CSS** | dark/light 主题、响应式 |
| 国际化 | **react-i18next** | 浏览器语言检测 + localStorage 缓存 |
| 构建 | **Vite** | 快速 HMR |
| 部署 | 嵌入 Rust 服务器 | `tower-http::services::ServeDir` |

## 路由

| 路径 | 内容 |
|------|------|
| `/` | 301 重定向到 `/ui/` |
| `/ui/` | 前端首页（搜索框 + 图谱可视化） |
| `/ui/index.html` | 静态入口 |

## 功能清单

### 顶部导航栏

| 组件 | 说明 |
|------|------|
| 图选择下拉框 | 调用 `GET /graphs` 列出所有图；选中后切换当前图 |
| 添加图 | 按钮弹出输入框，输入图名 + time_travel 选项，调 `POST /graphs` |
| 删除图 | 确认弹窗后调 `DELETE /graphs/{name}` |
| Time travel 时间线 | 双端滑块（graph 创建时间 → 当前时间），拖动后设置 `query_time` 并刷新图谱 |
| Compact 按钮 | 弹出模态框选择时间点（预设 1天/7天/30天/自定义），调 `POST /compact` |
| Extract 按钮 | 弹出模态框：拖入或粘贴 Markdown，调 `POST /extract`，显示进度 |
| 语言切换 | 中文 / English 按钮切换 |
| 主题切换 | 🌙/☀️ |

### 搜索

| 功能 | 说明 |
|------|------|
| 关键字搜索（默认） | 多个关键字用空格分隔，调 `keywordSearch` |
| 全文搜索 | 调 `semanticSearch`，LLM 提取关键字后搜索 |
| 搜索切换 | 按钮在 关键字/全文 间切换 |
| 高级搜索 | 展开面板：顶点标签过滤、边标签过滤、属性 key-value 过滤 |
| 首次进入 | 大搜索框鼓励输入长文本进行全文搜索 |

### 图谱可视化

| 功能 | 说明 |
|------|------|
| 力导向布局 | vis-network 物理仿真自动布局 |
| 双击顶点 | 以该顶点为起点，depth=1 遍历 `out(label?)`，结果合并到图谱 |
| 单击顶点 | 右侧属性面板展示该顶点属性 |
| 单击边 | 右侧属性面板展示该边属性 |
| 拖拽 | 自由拖拽节点，物理引擎重新平衡 |
| 缩放 | 鼠标滚轮缩放 |

### 属性编辑

| 功能 | 说明 |
|------|------|
| 查看属性 | 单击顶点/边后右侧面板展示所有属性 |
| 编辑属性 | 点击 [编辑] 按钮使属性值变为输入框 |
| 保存 | 提交到 `POST /gremlin` 更新属性 |
| 关闭 | 右侧面板可关闭 |

### 语言 (i18n)

| 功能 | 说明 |
|------|------|
| 自动检测 | `navigator.language` 检测，支持 `zh-CN` / `en-US` |
| 人工切换 | 顶栏按钮切换 |
| 缓存 | 切换后写入 `localStorage('lang')`，下次加载优先读取 |
| 覆盖范围 | 所有界面文案 + vis-network 节点中文标签显示 |

### 主题

| 功能 | 说明 |
|------|------|
| 深色模式（默认） | 深灰背景 `bg-gray-900` + 亮色节点 |
| 浅色模式 | 白背景 + 深色节点 |
| 切换 | 顶栏 🌙/☀️ 按钮 |
| 缓存 | 写入 `localStorage('theme')` |
